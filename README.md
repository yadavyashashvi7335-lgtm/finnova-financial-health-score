# 🏦 FinNova — Financial Health Score

> IDBI Innovate 2026 Hackathon Submission | Track: Financial Health Score

## 📌 Problem Statement
Most banking customers don't understand their financial health beyond a basic credit score. FinNova solves this by generating a multi-dimensional Financial Health Score (0–100) with an AI-powered chatbot that explains the score in plain language and gives personalized improvement tips.

## 🚀 Features
- ✅ ML-based Financial Health Score (0–100) using XGBoost
- ✅ AI Chatbot (LangChain + Groq) that explains your score
- ✅ Spending breakdown dashboard (Streamlit)
- ✅ Risk flag detection
- ✅ IDBI Bank product recommendations

## 🛠 Tech Stack
- Frontend: Streamlit
- Backend: FastAPI
- ML Model: XGBoost, Scikit-learn, Pandas
- AI Layer: LangChain + Groq (LLaMA 3 70B)
- Database: SQLite
- Deployment: Streamlit Cloud / Render
